# Práctica Advanced SQL

Este repositorio contiene la resolución de una práctica de **SQL avanzado**, organizada en distintos ejercicios independientes. Cada ejercicio se encuentra en su propio archivo `.sql`, lo que facilita la lectura, corrección y reutilización del código.

## Descripción

- Cada archivo `.sql` corresponde a un ejercicio concreto de la práctica.
- Los scripts están numerados siguiendo el orden del enunciado.
- El archivo `1.pdf` contiene el enunciado original o material de referencia.
- Los ejercicios trabajan consultas de SQL avanzado, incluyendo:
  - Subconsultas
  - JOINs complejos
  - Funciones de agregación
  - Consultas anidadas
  - GROUP BY y HAVING
  - Otras técnicas avanzadas de SQL

## Requisitos

Para ejecutar los scripts es necesario disponer de:
- Un sistema gestor de bases de datos compatible con SQL estándar (MySQL, PostgreSQL, SQLite, etc.).
- Acceso a la base de datos indicada en el enunciado de la práctica.

## Uso

1. Abrir el gestor de bases de datos elegido.
2. Seleccionar la base de datos correspondiente.
3. Ejecutar los archivos `.sql` de forma individual según el ejercicio deseado.

## Observaciones

- Los scripts son independientes entre sí, salvo que el enunciado indique lo contrario.
- Se recomienda revisar el archivo PDF antes de ejecutar los ejercicios para comprender el contexto de cada consulta.
